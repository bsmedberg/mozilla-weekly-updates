Markdown Details

In general, status updates use markdown syntax. See daringfireball.net for details about Markdown.

The following extras are supported:

  • Automatic linkifying: Anything that looks like a link starting with http: or https: will automatically be made into a link.
  • Automatic bug linkifying. Things like "bug 12345" will be auto-linked to bugzilla.mozilla.org.
  • Cuddled lists: a new line that starts with a start will be made into a list (normal markdown requires an intermediate blank line).

Sometimes, the extensions will cause problems, or there may be bugs in the python-markdown2 parsing module. If the parser throws an exception, this site falls back to showing the content as plain text.

If you encounter a bad parsing behavior, please copy the exact input text and paste into a github issue.
